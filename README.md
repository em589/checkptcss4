# MyStore - Responsive Store Landing Page

A responsive and elegant store landing page built with **HTML**, **CSS**, and **vanilla JavaScript**. This page showcases products in a modern grid layout and includes a hero section, navigation menu, product section, about section, and footer.

---

## Demo

You can view a live demo by opening `index.html` in your browser.

---

## Features

- Fully responsive design for desktop, tablet, and mobile.
- Hamburger menu for mobile navigation.
- Hero section with a call-to-action button.
- Product showcase with cards including image, description, price, and "Shop Now" button.
- About section describing the store.
- Simple and clean footer.
- Interactive mobile menu using JavaScript.

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (vanilla)
